# named-regexp-debs
Build Debian Package for Named capture groups for Java regular expressions

**Build dependencies**

- debhelper (>= 9)
- cdbs
- default-jdk
- maven-debian-helper (>= 1.5)

**Tested**

- Debian wheezy
- Debian jessie
